# ESP32 Cpp-Lesson
ESP32 C++ 教程，基于ESP32-Arduino-Framework应用开发框架。

[ESP32学习套件](https://www.xpstem.com/product/esp32-study-suit)

![ESP32学习套件](assets/study-suit-s.jpg)

### Unit0 介绍
* 课程介绍

### Unit1 基础知识
* 1-1 开发环境
* 1-2 C++ 基础知识
* 1-3 ESP32 Arduino开发框架

### Unit2 LED
* 2-1 点亮LED
* 2-2 RGB三色LED
* 2-3 WS2812灯珠（Adafruit_NeoPixel库）

### Unit3 输入
* 3-1 按键输入（按键信号转换、OneButton库、匿名函数、硬件与软件的推荐交互流程）
* 3-2 触摸感应输入

### Unit4 传感器
* 4-1 触摸传感器（数字量传感器使用、智能指针、C++标准集集合使用、传感器的推荐交互流程）
* 4-2 旋转电位器模块（模拟量传感器使用）
* 4-3 温湿度传感器（自定义传感器类、DHT11库、继承）
* 4-4 红外遥控（IRremote库）

### Unit5 执行器件
* 5-1 继电器
* 5-2 舵机控制

### Unit6 定时器和多线程
* 6-1 定时器（FreeRTOS定时器和硬件定时器）
* 6-2 多任务
* 6-3 任务间通信
* 6-4 共享资源访问

### Unit7 显示器件
* 7-1 7段式显示屏（TM1650库）
* 7-2 OLED显示屏（U8G2库）
* 7-3 TFT-LCD显示屏（TFT_eSPI库）

### Unit8 网络
* 8-1 WiFi连接
* 8-2 Web服务（WebServer库）
* 8-3 在网页上控制设备

### Unit9 存储
* 9-1 Flash存储
* 9-2 SD卡存储（SD）
